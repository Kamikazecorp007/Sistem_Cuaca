# Sistem_Cuaca
Website sederhana untuk mencari informasi negara di seluruh dunia sekaligus melihat kondisi cuaca terkini di ibu kotanya secara real-time.

Fitur Utama
Pencarian Negara: Mencari data negara menggunakan REST Countries API.

Info Cuaca Real-time: Menampilkan suhu dan deskripsi cuaca ibu kota menggunakan OpenWeatherMap API.

Desain Responsif: Tampilan kartu (card) yang rapi dengan CSS Grid dan Flexbox.

Interaktif: Dilengkapi dengan loading spinner dan fitur tekan "Enter" untuk mencari.
-------------------------------------------------
eknologi yang Digunakan
HTML5 & CSS3: Untuk struktur dan desain (Custom CSS).

JavaScript (ES6+): Menggunakan Fetch API, Async/Await, dan manipulasi DOM.

APIs:

REST Countries API

OpenWeatherMap API
------------------------------------------------
Jika cuaca muncul sebagai "Gagal memuat" atau "Invalid Key", harap tunggu 30-120 menit setelah pembuatan API Key baru karena adanya masa aktivasi dari server OpenWeather.

Pencarian disarankan menggunakan Bahasa Inggris (contoh: Japan, Germany, Indonesia).
